# Training Log

A personal bodyweight fitness tracker, served live with GitHub Pages.

Built around a 4-day-per-week bodyweight plan and four focus areas: strength &
muscle, cardio & endurance, body composition, and general health.

## Status

Scaffold is live. The full tracker UI is in progress.

## How updates work

This site is served by GitHub Pages from the `main` branch. Any commit that
changes `index.html` updates the live site automatically, usually within a
minute (occasionally up to ten).
